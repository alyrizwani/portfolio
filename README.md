# Ali Raza - Professional Portfolio

A modern, responsive portfolio website showcasing my work as a WordPress Developer and Graphic Designer.

## 🎨 Features

- **Responsive Design** - Works seamlessly on mobile, tablet, and desktop devices
- **Dark Theme** - Professional dark mode with blue/purple gradient accents
- **Smooth Navigation** - Mobile-friendly hamburger menu and smooth scrolling
- **Interactive Elements** - Hover animations, scroll effects, and smooth transitions
- **Profile Section** - Beautiful hero section with profile image that overflows container
- **About Me** - Personal background and introduction
- **Skills Showcase** - Six skill cards highlighting expertise areas
- **Projects Display** - Featured projects with links to live websites
- **Timeline** - Professional experience and education history
- **Contact Section** - Easy ways to get in touch
- **Social Links** - Direct links to email, WhatsApp, LinkedIn, and GitHub

## 🚀 Quick Start

1. **Download Files** - Ensure you have all three files in the same directory:
   - `index.html` - Main HTML structure
   - `styles.css` - Styling and responsive design
   - `script.js` - Interactive functionality
   - `self.jpeg` - Your profile photo

2. **Open in Browser** - Simply double-click `index.html` or open it in your preferred web browser

3. **No Setup Required** - The portfolio is a standalone static site with no dependencies

## 💻 Customization

### Update Profile Image
Replace `self.jpeg` with your own photo and update the filename in `index.html`:
```html
<img src="your-image.jpg" alt="Your Name" class="profile-img">
```

### Change Color Scheme
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary: #667eea;
    --secondary: #764ba2;
    --accent: #a78bfa;
    /* ... more variables */
}
```
